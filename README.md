# 🗂️ Respaldo Mensual

Sistema automatizado con interfaz gráfica para realizar copias de seguridad de archivos basado en rangos de fechas, optimizando la organización y protección de la información.

---

## 📌 Descripción

Este proyecto permite identificar, filtrar y respaldar archivos dentro de un rango de fechas específico (primer dia del mes, al ultimo) mediante una interfaz gráfica amigable.  
Incluye automatización de respaldos, facilitando la gestión de información sin intervención constante del usuario.

---

## 🚀 Funcionalidades

- 🖥️ Interfaz gráfica amigable e intuitiva
- 📅 Filtrado de archivos por rango de fechas
- 📁 Respaldo de carpetas clave:
  - Escritorio
  - Documentos
  - Descargas
  - Imágenes
  - Correos
- 🗂️ Organización automática de archivos respaldados
- 🔄 Conservación de la estructura original
- ⏰ Programación automática de respaldos
- ⚡ Ejecución rápida y eficiente

---

## ▶️ Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/MaryAlva09/Respaldo-Mensual.git
   Ejecuta la aplicación
2. Selecciona el rango de fechas
3. Configura (opcional) la automatización del respaldo
4. El sistema generará automáticamente el respaldo organizado
