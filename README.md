# Respaldo Mensual

Script para realizar respaldos automáticos de archivos por rango de fechas.

## 🚀 Funcionalidades
- Respaldo de Documentos, Imágenes, Descargas , Escritorio y Correos
- Filtrado por fecha
- Organización automática

## 🛠️ Tecnologías
- Python, XML

## ▶️ Cómo usar
1.- Ejecuta el instalador
2.- Sigue los pasos del instaldor
3.- Selecciona la configuracion en la ventana de configuracion
4.- Se instala tarea automatica
